# Crosshair overlay (Python)

Simples overlay de mira com PyQt5. Desenha dois círculos concêntricos (apenas contorno) sobre um fundo transparente.

Dependências
- Python 3.7+
- PyQt5
- pynput

Instalação

```bash
pip install -r requirements.txt
```

Execução

```bash
python cr2.py
```

Uso
- A mira fica fixa no centro da tela.
- Pressione `F1` para alternar entre as cores predefinidas (branco, vermelho, verde, azul).
- Feche a janela com `F2` encerrar o aplicativo.
